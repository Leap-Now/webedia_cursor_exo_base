# Project Specification

Build the official website for the movie "La femme de ménage" using the Eleventy (11ty) static site generator.

## Layout

Refer to `@/.specs/layout.md` for the primary site and component layout structure. Follow this to ensure visual and structural consistency.

## Pages

Implement these main pages:

- **Countdown Page**  
  - Reference details: `@/.specs/countdown.md`
  - Purpose: Show a countdown timer to the theatrical release date.
  - Content: Include movie branding (title, subtitle, logo) and a featured video section as specified.

- **Showtimes Page**  
  - Reference details: `@/.specs/showtimes.md`
  - Purpose: List cinema locations and movie showtimes for the film.
  - Content: Display information as described in the showtimes spec.

The site should provide seamless navigation between these pages, feeling unified and app-like (single-page app navigation style).

## Assets and Logos

- Use the Allociné logo (`@/public/assets/allocine.svg`) wherever Allociné branding, icons, badges, or attribution are required.
- Use the "LA FEMME DE MENAGE" logo (`@/public/assets/logo-lfdm.svg`) for main site branding, headers, and featured sections.

## General/UI Requirements

- Apply a consistent visual style across all UI elements and pages, strictly following the provided visual specs.
- Ensure full responsiveness: all components/layouts must work well on mobile, tablet, and desktop.
- Use best practices associated with the specified technology stack.
- When in doubt, prioritize consistency with the design assets and the referenced detailed page specs.
